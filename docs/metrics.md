# Operational Metrics

Track a few practical metrics to prove this system helps.

## 1) Context retrieval time
- Definition: time to answer recurring operational question
- Target: down 30%+ over 30 days

## 2) Duplicate analysis rate
- Definition: repeated analyses of same topic per month
- Target: down steadily as compiled corpus matures

## 3) Stale compiled ratio
- Definition: compiled pages older than freshness threshold / total compiled pages
- Target: <20% for active streams

## 4) Lineage coverage
- Definition: compiled pages with valid raw/run links / total compiled pages
- Target: 100% for high-impact pages
