# Anti-Patterns to Avoid

1. **Compiled without lineage**
   - Synthesis with no raw references = low trust.

2. **Raw without metadata**
   - Untraceable inputs kill future reuse.

3. **Over-automation without review**
   - Fast drift is still drift.

4. **Stale summaries as source-of-truth**
   - Old compiled pages should not drive current decisions alone.

5. **One giant stream for everything**
   - Use focused streams to preserve query precision.

6. **Publishing internal differentiators**
   - Share mechanics, not confidential product strategy.
