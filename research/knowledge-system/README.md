# Knowledge System (Raw → Compiled)

A simple operating loop for turning fragmented notes into queryable organizational memory.

## Loop

1. Ingest source snapshots into `raw/`
2. Compile to reusable docs in `compiled/`
3. Maintain index and lineage links
4. Query corpus first for non-trivial asks
5. Write outcomes back into the system

## Streams

Start small (1-3 streams):
- product
- operations
- maintainer-work

## Cadence

- Daily: ingest + compile for active streams
- Tue/Fri: health checks

## Success criteria (30 days)

- Faster context retrieval
- Fewer repeated analyses
- Clear evidence lineage for important decisions
